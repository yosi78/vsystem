# 💻 מערכת הזמנת חדר מחשבים

אפליקציית ווב מתקדמת לניהול הזמנות חדר מחשבים בבית ספר, עם תמיכה ב-Firebase לעדכון בזמן אמת.

---

## 📁 קבצים במערכת

1. **index.html** - מבנה האפליקציה (HTML)
2. **style.css** - עיצוב מודרני מותאם מובייל
3. **script.js** - לוגיקה ופונקציונליות עם Firebase

---

## 🚀 הוראות התקנה

### שלב 1: הכנת Firebase

#### 1.1 יצירת פרויקט
1. היכנסי ל-**[Firebase Console](https://console.firebase.google.com/)**
2. לחצי על **"Add project"** (הוסף פרויקט)
3. תני שם לפרויקט (למשל: "computer-room-booking")
4. עברי את שלבי היצירה

#### 1.2 הוספת אפליקציית Web
1. בדף הראשי של הפרויקט, לחצי על **Web** (</>) 
2. תני כינוי לאפליקציה (למשל: "Computer Room App")
3. **העתיקי** את פרטי ההגדרה שמופיעים (תצטרכי אותם בשלב הבא)

#### 1.3 הפעלת Realtime Database
1. בתפריט הצדדי, לחצי על **"Realtime Database"**
2. לחצי **"Create Database"**
3. בחרי מיקום (בדרך כלל europe-west1)
4. בחרי **"Start in test mode"** (לפיתוח)
5. לחצי **"Enable"**

#### 1.4 הגדרת כללי אבטחה (לפיתוח בלבד)
במסך ה-Database, לחצי על טאב **"Rules"** ושני את הכללים ל:

```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```

**⚠️ חשוב:** כללים אלה מתאימים רק לפיתוח! לסביבת ייצור, יש להגדיר כללי אבטחה מתאימים.

---

### שלב 2: הגדרת הקוד

1. פתחי את הקובץ **`script.js`**
2. מצאי את השורות 2-10 (הגדרות Firebase)
3. החליפי את הערכים בפרטים שהעתקת מ-Firebase:

```javascript
const firebaseConfig = {
    apiKey: "AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXX",
    authDomain: "your-project.firebaseapp.com",
    databaseURL: "https://your-project-default-rtdb.firebaseio.com",
    projectId: "your-project",
    storageBucket: "your-project.appspot.com",
    messagingSenderId: "123456789",
    appId: "1:123456789:web:xxxxxxxxxxxxx"
};
```

---

### שלב 3: העלאה לאירוח

#### אופציה 1: Firebase Hosting (מומלץ)
```bash
npm install -g firebase-tools
firebase login
firebase init hosting
# בחרי את הפרויקט שיצרת
# הגדירי את public directory ל-"."
firebase deploy
```

#### אופציה 2: Netlify
1. צרי חשבון ב-[Netlify](https://www.netlify.com)
2. גררי את 3 הקבצים לאזור ה-drag & drop
3. האתר יועלה אוטומטית

#### אופציה 3: Vercel / GitHub Pages
פעלי לפי ההוראות של הפלטפורמה הנבחרת.
