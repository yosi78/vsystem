📧 הגדרת EmailJS עם Gmail הוראות הפעלה - שליחת מייל אוטומטי למנהל

1
יצירת חשבון EmailJS
  1. היכנס לאתר emailjs.com
  2. לחץ על Sign Up והירשם עם כל מייל שתרצה
  3. אמת את החשבון דרך המייל שקיבלת
2
חיבור Gmail וקבלת Service ID
  1. בתפריט הצדדי לחץ על Email Services
  2. לחץ Add New Service
  3. בחר Gmail
  4. לחץ Connect Account ← בחר את חשבון Gmail שלך ← אשר גישה
  5. לחץ Create Service
  6. 📋 העתק את Service ID שנוצר (נראה כך: service_xxxxxxx)
3
יצירת תבנית מייל וקבלת Template ID
  1. בתפריט לחץ על Email Templates
  2. לחץ Create New Template
  3. EmailJS פותח תבנית ברירת מחדל עם שדות לדוגמה — מחק את כל התוכן הקיים ב-Subject וב-Content והחלף בשלנו:
שדה בתבנית מה לרשום
To Email בחר מהרשימה את כתובת ה-Gmail שחיברת בשלב 2
Subject מחק מה שיש ← הדבק: הזמנה חדשה - {{teacher}}
Content מחק מה שיש ← הדבק את הטקסט מהמלבן הכהה למטה
💡 המשתנים כמו {{teacher_name}} לא צריך להגדיר מראש — פשוט כתוב אותם בתוך התבנית ו-EmailJS יזהה אותם אוטומטית.
מורה: {{teacher}} כיתה: {{kita}} תאריך: {{date}} פריטים שהוזמנו: {{items}}
  1. לחץ Save
  2. 📋 העתק את Template ID (נראה כך: template_xxxxxxx)
4
קבלת Public Key
  1. לחץ על שמך בפינה הימנית העליונה ← Account
  2. לחץ על General
  3. תחת API Keys — 📋 העתק את Public Key
5
הזנה במסך ההגדרות במערכת

היכנס למערכת כמנהל ← לחץ על טאב 🔔 הגדרות ומלא:

שדה מה להכניס
📮 כתובת מייל לקבלת התראות כתובת ה-Gmail שלך
Public Key מה שהעתקת בשלב 4
Service ID מה שהעתקת בשלב 2
Template ID מה שהעתקת בשלב 3

לחץ 💾 שמור הגדרות — וזהו, המיילים יתחילו להגיע אוטומטית!

⚠️ שים לב: כתובת הנמען היא ה-Gmail שחיברת בשלב 2. אם תרצה לשנות את הכתובת בעתיד — חבר שירות Gmail חדש וערוך את שדה To Email בתבנית.
